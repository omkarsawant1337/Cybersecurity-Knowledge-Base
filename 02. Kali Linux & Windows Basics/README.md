## 📂 Topics Covered

### 🐧 Essential Linux Commands

Build a strong command-line foundation by learning the most frequently used Linux commands.

Topics include:

- File Management & Directory Navigation
- User Management & Package Management
- Process Management & System Monitoring
- Networking Commands
- File Searching, Compression, SSH, SCP
- Disk Usage & Text Processing
- Command History, File Comparison, Symbolic Links

### ⚡ Bash Scripting

Learn how to automate repetitive tasks using Bash.

Topics include:

- Bash Basics, Shell Scripts, Variables, User Input
- Conditional Statements, Loops, Functions, Arrays
- Command Substitution & Exit Codes
- File Handling & Script Arguments
- Automation — Network Scanning Scripts, Recon Automation
- Error Handling

### 🔐 Linux File Permissions

Understand Linux security through ownership and permissions.

Topics include:

- `ls -l` Output, Read/Write/Execute Permissions
- Numeric & Symbolic Permissions
- `chmod`, `chown`, `chgrp`
- SUID, SGID, Sticky Bit
- File Ownership & Directory Permissions
- Permission Inheritance & Privilege Escalation Concepts

### 🪟 Windows File Permissions

Learn Windows NTFS permissions and Access Control Lists (ACLs).

Topics include:

- NTFS Permissions, ACLs, DACL, SACL
- Security Identifiers (SID)
- Ownership & Permission Inheritance
- `icacls`, Access Control Entries, Security Descriptors
- Effective Permissions & Permission Auditing
- Active Directory Permission Concepts

### ⏰ Cron Jobs

Automate Linux tasks using Cron.

Topics include:

- Cron Daemon, Crontab, Cron Syntax
- Scheduling Jobs — System Cron, User Cron
- Log Rotation & Automated Backups
- Scheduled Recon & Task Automation
- Cron Security & Cron Privilege Escalation

### 💻 Top 200 Windows Commands

Master commonly used Windows Command Prompt utilities.

Topics include:

- File Operations & Network Commands
- System Information & Process Management
- User Management & Service Management
- Disk Management & Environment Variables
- Networking Utilities & Troubleshooting Commands
- Security & Administrative Commands

### ⚙ PowerShell Basics

Learn Windows automation and administration using PowerShell.

Topics include:

- PowerShell Architecture & Cmdlets
- Variables, Objects, Pipeline
- Loops, Functions, Arrays, Hashtables
- File Operations, Services, Networking
- Remote Management & Automation Scripts
- Windows Administration

### 📝 Windows Registry

Understand the Windows Registry and its role in system configuration and security.

Topics include:

- Registry Structure, Root Hives, Registry Keys & Values
- Registry Data Types & `regedit`
- Registry Commands & Startup Locations
- Persistence & Malware Persistence
- Registry Security & Forensics

### ⚙ Windows Services

Explore Windows background services and service management.

Topics include:

- Service Control Manager (SCM) & Service Accounts
- Service Startup Types & Configuration
- Service Permissions — `sc.exe`, `Get-Service`, `New-Service`
- Service Enumeration & Windows Internals
- Privilege Escalation & Service Security

## 🚀 Why Learn These Topics?

Linux and Windows form the backbone of modern cybersecurity. Ethical Hackers, Penetration Testers, SOC Analysts, Red Teamers, and Blue Teamers rely on these operating systems daily.

By mastering these fundamentals, you'll be able to:

- Navigate Linux confidently
- Automate repetitive security tasks
- Understand Linux and Windows permissions
- Secure operating systems
- Perform privilege escalation assessments
- Analyze Windows internals
- Manage Windows services and the registry
- Use PowerShell effectively
- Work efficiently with Kali Linux
- Build a strong foundation for penetration testing and Active Directory security
