## 📂 Topics Covered

### 🛡 Foundations of Ethical Hacking

Learn the fundamentals of ethical hacking and how penetration testers approach security assessments.

Topics include:

- Introduction to Hacking & Ethical Hacking
- Types of Hackers & Cybersecurity Roles
- Ethical Hacking Methodology
- Red Team, Blue Team, Purple Team
- Information Gathering, Vulnerability Assessment, Exploitation, Post Exploitation
- Security Reporting
- Common Security Threats & Essential Hacking Tools

### 🌐 Networking for Ethical Hackers

Understand how networks operate from both an administrator's and attacker's perspective.

Topics include:

- Network & Internet Fundamentals
- OSI Model & TCP/IP Model
- IP Addressing & MAC Address
- Ports & Services, Client-Server Model
- Inbound & Outbound Traffic, Subnetting
- Critical Network Protocols, Routing Basics
- Network Security Concepts

### 🌍 Internet Protocol Suite (TCP/IP)

Explore the protocol suite that powers the modern Internet.

Topics include:

- TCP/IP Architecture — Application, Transport, Internet, Link Layers
- TCP, UDP, QUIC
- HTTP/HTTPS, DNS, SMTP, SSH, DHCP, ICMP
- Routing & Packet Flow
- Encapsulation

### 🌐 Web Architecture Fundamentals

Learn how modern web applications communicate and are structured behind the scenes.

Topics include:

- Client-Server Architecture
- HTTP / HTTPS, Request & Response Cycle
- HTTP Status Codes & HTTP Methods
- Request & Response Headers
- Cookies & Sessions
- Authentication vs. Authorization
- Web Servers & Reverse Proxies
- APIs & REST Architecture

### 🖧 Network Topologies

Learn how devices are connected in modern computer networks.

Topics include:

- Physical vs. Logical Topology
- Bus, Ring, Star, Mesh, Tree, Hybrid Topologies
- Performance, Scalability, Fault Tolerance
- Security Considerations

### ⚛ Periodic Table of Cybersecurity

A visual overview of the cybersecurity ecosystem.

Topics include:

- Threats, Vulnerabilities, Security Tools
- Frameworks & Security Standards
- Security Concepts & Defensive Measures
- Security Roles & Common Protocols
- Industry Terminology

### 🌐 The Seven Layers of the OSI Model

Master one of the most important networking models used in cybersecurity.

Topics include:

- Physical, Data Link, Network, Transport, Session, Presentation, Application Layers
- Layer Responsibilities & Protocols by Layer
- Devices by Layer
- Encapsulation & Decapsulation
- Real-World Examples

### 🔒 CIA Triad

Understand the three fundamental pillars of information security.

Topics include:

- Confidentiality, Integrity, Availability
- Data Protection & Access Control
- Encryption & Hashing
- Redundancy & Backup Strategies
- Security Best Practices

### ⚔ Cyber Kill Chain

Learn how cyber attacks progress through each stage.

Topics include:

- Reconnaissance → Weaponization → Delivery → Exploitation → Installation → Command & Control (C2) → Actions on Objectives
- Detection Opportunities & Defensive Strategies at each stage

### 🎯 MITRE ATT&CK Framework

Understand real-world adversary behaviors using the MITRE ATT&CK knowledge base.

Topics include:

- ATT&CK Matrix — Tactics, Techniques, Sub-techniques
- Enterprise ATT&CK: Initial Access, Persistence, Privilege Escalation, Defense Evasion, Credential Access, Discovery, Lateral Movement, Command & Control, Exfiltration, Impact

### 📦 Protocol Data Units (PDUs)

Learn how data is encapsulated and transmitted across networks.

Topics include:

- Ethernet Frames, IP Packets, TCP Segments, UDP Datagrams, ICMP Messages
- Encapsulation & Decapsulation
- Headers & Payloads
- Frame Structure & Packet Analysis

### 🚦 Differentiated Services Code Point (DSCP)

Learn how modern networks classify and prioritize traffic for Quality of Service (QoS).

Topics include:

- Quality of Service (QoS) Fundamentals
- DiffServ (Differentiated Services) Architecture
- The DSCP Field in the IP Header
- Per-Hop Behaviors (PHBs) — Expedited Forwarding, Assured Forwarding, Class Selector
- DSCP Values & Common Markings (Voice, Video, Data)
- Traffic Classification & Marking
- Traffic Shaping, Policing, and Congestion Management
- DSCP vs. ToS (Type of Service)
- Real-World QoS Use Cases (VoIP, Video Conferencing)

### 🔢 Common Networking & Security Ports

Master the ports every cybersecurity professional should know.

Topics include:

- Well-Known Ports, Registered Ports, Dynamic Ports
- TCP Ports vs. UDP Ports
- FTP, SSH, HTTP, HTTPS, DNS, SMB, RDP, LDAP, Kerberos, SNMP
- SQL Services & Email Protocols

### 💀 Types of Cyber Attacks

Understand the most common attacks seen in modern environments.

Topics include:

- Phishing & Spear Phishing
- Malware & Ransomware
- SQL Injection, XSS, CSRF, Command Injection, Buffer Overflow
- Man-in-the-Middle (MITM), DoS/DDoS
- Password Attacks, Insider Threats, Social Engineering
- Zero-Day Attacks, Drive-by Downloads, Botnets, Cryptojacking

## 🚀 Why Learn These Topics?

Every cybersecurity domain — whether Penetration Testing, SOC Analysis, Threat Hunting, Digital Forensics, Cloud Security, or Red Teaming — relies on strong networking and security fundamentals.

By mastering these concepts, you'll be able to:

- Understand how networks communicate
- Analyze packets and protocols
- Identify common attack vectors
- Interpret attacker behavior using MITRE ATT&CK
- Understand the Cyber Kill Chain
- Build a strong networking foundation
- Perform effective reconnaissance
- Communicate confidently during interviews
- Prepare for certifications like Security+, CEH, PNPT, eJPT, and OSCP
