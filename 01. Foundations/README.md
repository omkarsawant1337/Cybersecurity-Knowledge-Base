## 📂 Topics Covered

### 🎯 Foundations of Ethical Hacking

Learn the core concepts behind ethical hacking and penetration testing.

Topics include:

- Introduction to Hacking & Ethical Hacking
- Types of Hackers — White Hat, Black Hat, Gray Hat, Script Kiddies
- Ethical Hacking Phases
- Red Team, Blue Team, Purple Team
- Common Pentesting Tools
- Information Security Threats & Risks
- Security Best Practices
- Kali Linux Introduction

### 🌐 Networking for Ethical Hackers

Understand how modern computer networks operate.

Topics include:

- Network & Internet Fundamentals
- LAN, WAN, MAN, PAN, WLAN
- IP Address, MAC Address, Gateway
- DNS, DHCP
- Routers, Switches, Hubs, Firewalls
- OSI Model & TCP/IP Model Overview
- Network Devices & Common Network Protocols

### 🌍 Internet Protocol Suite (TCP/IP)

Master the protocol suite powering the modern Internet.

Topics include:

- TCP/IP Architecture — Application, Transport, Internet, Link Layers
- TCP, UDP, QUIC
- HTTP, HTTPS, DNS, SMTP, IMAP, POP3, FTP, SSH, DHCP, SNMP, RTP, TLS
- Port Numbers & Socket Communication
- Flow Control & Congestion Control
- Three-Way Handshake

### 🌐 Web Architecture Fundamentals

Learn how modern web applications work behind the scenes.

Topics include:

- Client-Server Architecture
- HTTP / HTTPS, Request & Response, Status Codes, HTTP Methods
- Request & Response Headers
- Cookies & Sessions
- Authentication vs. Authorization
- Web Servers & Reverse Proxies
- APIs & REST Architecture

### 🌍 Network Topologies

Understand how networks are physically and logically designed.

Topics include:

- Physical vs. Logical Topology
- Bus, Ring, Star, Mesh, Tree, Hybrid Topologies
- Network Scalability, Fault Tolerance, Redundancy
- Enterprise Network Design

### 🧪 Periodic Table of Cybersecurity

A complete reference covering the cybersecurity ecosystem.

**Threats**
Phishing · Malware · Ransomware · XSS · DDoS · Spyware · Trojan · Rootkits · Keyloggers

**Vulnerabilities**
Unpatched Software · Open Ports · Broken Authentication · Privilege Escalation · CSRF · Clickjacking · Misconfigurations · Sensitive Data Exposure · Buffer Overflow · XXE · Weak Security Controls · Remote Code Execution

**Tools**
Nmap · Burp Suite · Metasploit · SQLMap · Hydra · Hashcat · Gobuster · Tcpdump · Aircrack-ng · Nikto · Snort · Kali Linux

**Frameworks**
MITRE ATT&CK · NIST · CIS Controls · OWASP · PCI DSS · COBIT · ISO 27001

### 🌐 OSI Model

Understand networking using the industry-standard seven-layer model.

Topics include:

- Layer 7 – Application → Layer 1 – Physical (full breakdown)
- Encapsulation & Decapsulation
- Protocol Mapping
- OSI vs. TCP/IP
- Layer Responsibilities & Common Protocols
- Real-World Networking Examples

### 🔐 CIA Triad

Learn the three pillars of information security.

Topics include:

- Confidentiality, Integrity, Availability
- Encryption, Access Control, Authentication, Authorization
- Hashing & Digital Signatures
- High Availability, Backup Strategies, Disaster Recovery
- Information Assurance

### ⚔ Cyber Kill Chain

Understand how cyber attacks progress from start to finish.

Topics include:

- Reconnaissance → Weaponization → Delivery → Exploitation → Installation → Command & Control (C2) → Actions on Objectives
- Defensive Opportunities & Detection Strategies at each stage
- Attack Lifecycle

### 🎯 MITRE ATT&CK Overview

Learn the industry-standard adversary behavior framework.

Topics include:

- Enterprise Matrix — Tactics, Techniques, Sub-techniques
- Full Tactic Chain: Reconnaissance, Initial Access, Execution, Persistence, Privilege Escalation, Defense Evasion, Credential Access, Discovery, Lateral Movement, Collection, Command & Control, Exfiltration, Impact
- Detection & Mitigations
- Threat Intelligence

## 🚀 Why Learn These Foundations?

Every cybersecurity specialization — whether Penetration Testing, Bug Bounty Hunting, Digital Forensics, Malware Analysis, Cloud Security, SOC Operations, or Red Teaming — depends on mastering these core concepts first.

These topics provide the technical knowledge needed to:

- Understand how networks communicate
- Analyze web applications
- Identify security weaknesses
- Understand attacker methodologies
- Interpret security frameworks
- Perform vulnerability assessments
- Build strong defensive strategies
- Prepare for advanced penetration testing
- Analyze incidents effectively
- Develop a security-first mindset
